To start a shared crossword, follow these steps.

  1. Pick one person to set things up. That player will do steps 2-3 while the others wait until step 4 to do anything.

  2. The player setting up the game should follow this link in a new tab and select a puzzle to do.

  3. Copy the URL from the URL bar of your browser and share it with the other players using the Gather chat.

  4. Once the URL has been shared, everyone should click here and enter the URL when prompted.
